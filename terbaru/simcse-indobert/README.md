# SentenceTransformer based on indobenchmark/indobert-base-p1

This is a [sentence-transformers](https://www.SBERT.net) model finetuned from [indobenchmark/indobert-base-p1](https://huggingface.co/indobenchmark/indobert-base-p1). It maps sentences & paragraphs to a 768-dimensional dense vector space and can be used for retrieval.

## Model Details

### Model Description
- **Model Type:** Sentence Transformer
- **Base model:** [indobenchmark/indobert-base-p1](https://huggingface.co/indobenchmark/indobert-base-p1) <!-- at revision c2cd0b51ddce6580eb35263b39b0a1e5fb0a39e2 -->
- **Maximum Sequence Length:** 512 tokens
- **Output Dimensionality:** 768 dimensions
- **Similarity Function:** Cosine Similarity
- **Supported Modality:** Text

```
SentenceTransformer(
  (0): Transformer({'transformer_task': 'feature-extraction', 'modality_config': {'text': {'method': 'forward', 'method_output_name': 'last_hidden_state'}}, 'module_output_name': 'token_embeddings', 'architecture': 'BertModel'})

)
```

## Usage

### Direct Usage (Sentence Transformers)

First install the Sentence Transformers library:

```bash
pip install -U sentence-transformers
```
Then you can load this model and run inference.
```python
from sentence_transformers import SentenceTransformer

# Download from the 🤗 Hub
model = SentenceTransformer("sentence_transformers_model_id")
# Run inference
sentences = [
    'aktifitas posyandu di desa gunggungan lor sama seperti aktifitas posyandu pada umumnya yaitu mencatat perkembangan anak mulai dari sebelum lahir hingga berumur maksimal 5 tahun proses pencatatan tiap perkembangan anak dicatat dan direkap pada buku kia dan catatan khusus petugas posyandu akan tetapi catatan tersebut sering kali hilang dan terselip dengan dokumen-dokumen lainnya baik buku kia atau catatan yang dipegang oleh petugas proses pencatatannya pun menggunakan proses pencatatan manual dan perekapan untuk pihak puskesmas juga manual sehingga membutuhkan waktu yang lama dan ruang penyimpanan kertas yang cukup banyak maka dari itu dibutuhkan aplikasi untuk proses pelaporan pada puskesmas yaitu menggunakan aplikasi web sebagai perekapan pelaporan dan statistik perkembangan',
    'aktifitas posyandu di desa gunggungan lor proses seperti aktifitas posyandu pada umumnya yaitu berumur perkembangan anak mulai dari sebelum lahir hingga mencatat maksimal 5 tahun sama pencatatan tiap perkembangan anak dicatat dan direkap pada buku kia dan catatan khusus petugas posyandu akan tetapi catatan tersebut sering kali hilang dan terselip dengan dokumen-dokumen lainnya baik buku kia atau catatan yang dipegang oleh petugas proses pencatatannya pun menggunakan proses pencatatan manual dan perekapan untuk pihak puskesmas juga manual sehingga membutuhkan waktu yang lama dan ruang penyimpanan kertas yang cukup banyak maka dari itu dibutuhkan aplikasi untuk proses pelaporan pada puskesmas yaitu menggunakan aplikasi web sebagai perekapan pelaporan dan statistik perkembangan',
    'pengendalian pressure digester temperature dan pada biogas',
]
embeddings = model.encode(sentences)
print(embeddings.shape)
# [3, 768]

# Get the similarity scores for the embeddings
similarities = model.similarity(embeddings, embeddings)
print(similarities)
# tensor([[ 1.0000,  0.9976, -0.0520],
#         [ 0.9976,  1.0000, -0.0476],
#         [-0.0520, -0.0476,  1.0000]])
```
